## Installation

1. Clone the repo

  ```
    https://github.com/AmauriMarcos/aurora.git
  ```

2. Install NPM packages

   ```
   npm install
   ```
   
3. Go to the project folder

   ```
    cd aurora
   ```

4. Start the project

   ```
   npm run dev
   ```
